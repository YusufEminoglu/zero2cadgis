Easily import DXF, Netcad NCZ/NCA, KML, KMZ, DGN, and GDB formats directly, with CAD collinear simplification, coordinate transformation, automatic labeling, database joins, and remembered import and export folders across sessions. The NCZ parser and NCZ-to-QGIS layer integration contain modified code from Jeomatik NCZ Reader, Copyright (C) 2026 Erdinç Örsan ÜNAL, under GPL-2.0-or-later. DWG import is tracked as a future enhancement.
